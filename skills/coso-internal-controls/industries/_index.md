## How industries are modeled

For each industry, the file follows the same shape:
1. Posture — typical SOX 404 filing status and COSO framework usage.
2. Entity-Level Control Profile — control environment, governance, risk assessment characteristics.
3. Deficiency Classification Impact — industry-specific severity factors.
4. Top Use Cases — links to use-cases/.
5. Pain Points — what practitioners get wrong.
