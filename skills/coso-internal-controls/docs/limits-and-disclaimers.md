# Limits and Disclaimers — coso-internal-controls

## 1. What this skill is NOT

- Not a substitute for professional judgment. Always have a qualified auditor review outputs.
- Not legal advice. SOX 404, SEC regulations, and PCAOB standards are interpreted by the SEC, PCAOB, and courts.
- Not a certification. Using the skill does not confer CPA, CIA, CISA, or any other credential.
- Not a replacement for an integrated audit. The skill helps produce artifacts; the auditor signs the opinion.

## 4. Known gaps

- COSO ICIF 2013 has 17 principles each with multiple Points of Focus; 71 are listed here (curated subset).
- PCAOB AS 2201 paragraph numbers were reorganized post-2007; verify against current PCAOB codification.
- Materiality thresholds are entity-specific; the skill cannot determine materiality for a specific engagement.
- Deficiency classification is professional judgment; the decision tree encodes the framework but does not replace the auditor's assessment.

## 5. Out-of-scope inputs

- Pure financial statement audit without ICFR (use GAAS audit skill).
- SOC 1/SOC 2 reporting (use AICPA SOC skill).
- Forensic investigation (use forensic skill, then COSO for control implications).
- Non-US entities not subject to SOX/PCAOB (adapt COSO principles to local framework).

## 6. When to refuse

- User asks for a categorical deficiency classification without system context.
- User provides contradictory information.
- User requests hallucinated paragraph numbers or control counts.

> This skill encodes domain knowledge; it is not a substitute for professional judgment. Always verify outputs against the cited authoritative source.
